# Задача 3 — Очередь через два стека

## Описание
Реализовать очередь, используя **два стека**

---

## Обязательная часть
Реализовать:
- `enqueue(x)`
- `dequeue()`
- `front()`

---

## Вариативная часть (выполнено задание №9)

Из предложенных вариативных заданий реализовано:

**9. Добавить проверку на переполнение (если переполнение — возвращать ошибку)**  
При добавлении элемента в очередь проверяется, не превышает ли новый размер максимально допустимый. Если очередь заполнена, метод `enqueue(x)` возвращает ошибку, а элемент не добавляется.

## Покрытие тестами

Все реализованные методы (`enqueue`, `dequeue`, `front`, а также проверка переполнения с возвратом ошибки) покрыты **unit-тестами**.
